This is my very first project, built to practice the basics of frontend development. It is a simple web-based drawing application created within a short time frame to learn how to manipulate the DOM and handle mouse events in Vanilla JavaScript.

## Features
* Drawing on screen by dynamically generating DOM elements
* Custom paintbrush color selection
* Changeable background color

## Stack
* HTML5
* CSS3
* JavaScript (Vanilla JS, DOM Manipulation, Event Listeners)

## Future Ideas
As my first project, I plan to improve it in the future by adding:
* Migrating the core logic to HTML5 Canvas API for better performance
* Brush size adjustment
* Quick canvas clearing
